# Reference Documentation

This directory contains stable lookup references for agents and maintainers.

Use this directory for:

- file-level code maps and entrypoints;
- permission and security boundary references;
- data model tables grouped by domain;
- compact maps that help route a task to the right code.

Do not put long workflow runbooks, dated reports, or append-only validation
ledgers here.

- [CODE_MAP_AND_ENTRYPOINTS.md](CODE_MAP_AND_ENTRYPOINTS.md)
- [PERMISSIONS_AND_SECURITY_BOUNDARIES.md](PERMISSIONS_AND_SECURITY_BOUNDARIES.md)
- [DATA_MODEL_ESSENTIALS.md](DATA_MODEL_ESSENTIALS.md)
