# Handoffs

This directory is reserved for committed handoff documents when the user
explicitly asks to prepare work for another agent or session.

Use this directory for:

- current problem statements;
- branch/worktree context;
- touched files and remaining plan;
- validation state and explicit warnings for the next agent.

Do not use this directory for ordinary dated reports, local scratch notes,
logs, screenshots, or private machine paths. Machine-local continuation notes
belong under ignored `.agent-run/`; durable rules belong in the relevant topic
docs.

If there is no active handoff request, this directory may contain only this
README.
