# Architecture Documentation

This directory defines the current CourseEval system structure and code
boundaries.

Use this directory for:

- backend package and import-boundary rules;
- major product flow maps with code anchors;
- configuration reference material;
- troubleshooting that routes symptoms to the right source files or runbooks;
- stable architecture facts that should outlive a single implementation round.

Do not put dated reports, temporary handoffs, local validation logs, or
append-only test ledgers here.
