# Operations Documentation

This directory documents deployment, bootstrap, and runtime operations.

Use this directory for:

- production layout and service configuration;
- nginx, systemd, environment templates, and deployment scripts;
- startup and first-admin/bootstrap behavior;
- operational procedures that deployers must follow.

Do not put local test triage, Playwright run histories, product feature guides,
or temporary handoffs here.

Primary entry points:

- [DEPLOYMENT_AND_OPERATIONS.md](DEPLOYMENT_AND_OPERATIONS.md)
- [ADMIN_BOOTSTRAP.md](ADMIN_BOOTSTRAP.md)
