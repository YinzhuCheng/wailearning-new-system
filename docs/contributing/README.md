# Contributing Documentation

This directory contains contributor workflow rules that apply across product
areas.

Use this directory for:

- Git and branch workflow;
- UTF-8 and PowerShell-safe text editing guidance;
- repository-wide contribution habits that are not product behavior, runtime
  architecture, deployment, or test evidence.

Do not put feature guides, frontend behavior notes, or test execution ledgers
here.

Primary entry points:

- [ENCODING_AND_MOJIBAKE_SAFETY.md](ENCODING_AND_MOJIBAKE_SAFETY.md)
- [GIT_WORKFLOW.md](GIT_WORKFLOW.md)
