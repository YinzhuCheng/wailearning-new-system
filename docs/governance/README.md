# Governance Documentation

This directory contains active repository governance, unresolved risks, and
decision records that should influence future work.

Use this directory for:

- known implementation-backed risks and ownership ambiguities;
- rules that prevent repeated agent mistakes;
- active governance notes that are not architecture, operations, product, or
  test-ledger material.

Do not put dated historical reports here. Use `docs/reports/` only when the
report remains useful evidence; otherwise keep the active rule in the relevant
topic folder.

Primary entry point:

- [known-issues-and-risks.md](known-issues-and-risks.md)
- [repository-governance.md](repository-governance.md)
- [agent-update-log.md](agent-update-log.md)
- [high-risk-path-metadata.json](high-risk-path-metadata.json)
