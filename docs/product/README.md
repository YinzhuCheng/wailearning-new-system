# Product Documentation

This directory documents CourseEval product behavior and domain concepts.

Use this directory for:

- LLM homework grading behavior and quotas;
- parent portal behavior;
- homework/submission content semantics;
- product-visible rules that are broader than one frontend component or one
  backend module.

Do not put implementation-only architecture maps, deployment procedures, or
test ledgers here.

- [LLM_HOMEWORK_GUIDE.md](LLM_HOMEWORK_GUIDE.md)
- [PARENT_PORTAL.md](PARENT_PORTAL.md)
